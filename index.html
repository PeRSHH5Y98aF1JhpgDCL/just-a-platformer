[INFO]
Width: 10
Height: 9
Left Click: Solid Block
Right Click: Empty Space
[CONTROL]
LMB & RMB: Place block
[shift] + LMB or RMB: Move player to cursor
MMB: Select block clicked for LMB
[shift] + MMB: Select block clicked for RMB
[,] & [.]: Cycle LMB block type
[shift] + [,] & [.]: Cycle RMB block type
[shift] + [arrow]: Increase level size
[ctrl] + [arrow]: Decrease level size
[S]: Restart from startpoint
[R]: Respawn from last checkpoint
[G]: Toggle god mode (ignore death + infinite jump)
[I]: Toggle info display
[C]: Toggle control display
[E]: Export level data
[shift] + [E]: Import level data
